# Opis projektu — Systemowe Projektowanie

## Nazwa projektu

CryptoTrack — system wizualizacji i analizy transakcji MetaMask Card

## Stos technologiczny

TypeScript, React 19, Next.js 15, Tailwind, NestJS, Fastify, PostgreSQL, Prisma, JWT, SIWE, REST API, rozszerzenie Chrome (Plasmo), Docker, pnpm, Turbo

## Wdrożenie

Docker, Docker Compose, Git, GitHub (wkrótce)

## Opis projektu

System do śledzenia wydatków kartą MetaMask Card: aplikacja web (dashboard, kategorie, analityka), API backend oraz rozszerzenie Chrome synchronizujące transakcje z panelu MetaMask. Logowanie e-mail/hasło lub portfelem Web3. CRUD: użytkownicy, transakcje, kategorie.
